# Jules API Full Reference

> [!IMPORTANT]
> The Jules API is currently in **alpha release**. Specifications, API keys, and definitions may change as the API moves toward stabilization.

The Jules API lets you programmatically access Jules's capabilities to automate and enhance your software development lifecycle.

---

## Service: jules.googleapis.com
**Base URL**: `https://jules.googleapis.com/v1alpha`  
**Authentication**: Required via `X-Goog-Api-Key` header.

---

## Resources

### Source (`v1alpha.sources`)
A code source connected to Jules.

| Field | Type | Description |
| :--- | :--- | :--- |
| `name` | `string` | The resource name: `sources/{source}` |
| `id` | `string` | Output only. The unique ID of the source. |
| `githubRepo` | `object` | Connection details for a GitHub repository. |
| `githubRepo.owner` | `string` | The owner of the repository. |
| `githubRepo.repo` | `string` | The name of the repository. |
| `githubRepo.isPrivate`| `boolean`| Whether the repository is private. |
| `githubRepo.defaultBranch`| `object` | The default branch info. Contains `displayName`. |
| `githubRepo.branches[]`| `array` | List of active branches. Items contain `displayName`. |

### Session (`v1alpha.sessions`)
A continuous unit of work (e.g., a feature request or bug fix).

| Field | Type | Description |
| :--- | :--- | :--- |
| `name` | `string` | The resource name: `sessions/{sessionId}` |
| `id` | `string` | Output only. The unique ID of the session. |
| `title` | `string` | A descriptive title for the session. |
| `prompt` | `string` | The initial task description provided to the agent. |
| `state` | `enum` | Current state ([see SessionState](#sessionstate)). |
| `createTime` | `timestamp` | When the session was created. |
| `updateTime` | `timestamp` | When the session was last updated. |
| `automationMode` | `enum` | Automation behavior ([see AutomationMode](#automationmode)). |
| `requirePlanApproval`| `boolean` | If true, waits for plan approval before execution. |
| `sourceContext` | `object` | The source and branch context for this session. |
| `sourceContext.source`| `string` | The source name, e.g., `sources/github/owner/repo`. |
| `sourceContext.githubRepoContext.startingBranch` | `string` | The base branch (Default: `main`). |
| `outputs` | `array` | Results generated by the session (e.g., Pull Requests). |

### Activity (`v1alpha.sessions.activities`)
A single event or unit of work within a session.

| Field | Type | Description |
| :--- | :--- | :--- |
| `name` | `string` | Resource name: `sessions/{sessionId}/activities/{activityId}` |
| `id` | `string` | Output only. Unique identifier for the activity. |
| `createTime` | `timestamp` | When the activity occurred. |
| `originator` | `enum` | Who initiated the activity (`system`, `agent`, or `user`). |
| `description` | `string` | Human-readable summary of the activity. |
| `userMessaged` | `object` | Content of a user message. |
| `agentMessaged` | `object` | Content of an agent message. |
| `planGenerated` | `object` | Details of a generated implementation plan. |
| `planApproved` | `object` | Records the approval of a plan. |
| `progressUpdated` | `object` | A status/progress update with a title and description. |
| `sessionCompleted`| `object` | Indicates session success. |
| `sessionFailed` | `object` | Indicates session failure and reason. |
| `artifacts` | `array` | Items produced by the activity (media, scripts, patches). |

## Endpoints

### Sources
- **List**: `GET /v1alpha/sources`
- **Get**: `GET /v1alpha/sources/{sourceName}`

### Sessions
- **List**: `GET /v1alpha/sessions`
- **Create**: `POST /v1alpha/sessions`
- **Get**: `GET /v1alpha/sessions/{sessionId}`
- **Send Message**: `POST /v1alpha/sessions/{sessionId}:sendMessage`
- **Approve Plan**: `POST /v1alpha/sessions/{sessionId}:approvePlan`

### Activities
- **List**: `GET /v1alpha/sessions/{sessionId}/activities`
- **Get**: `GET /v1alpha/sessions/{sessionId}/activities/{activityId}`

## Enums & Types

### SessionState
`QUEUED`, `PLANNING`, `AWAITING_PLAN_APPROVAL`, `AWAITING_USER_FEEDBACK`, `IN_PROGRESS`, `PAUSED`, `COMPLETED`, `FAILED`

### AutomationMode
- `NONE`: No automatic actions (Default).
- `AUTO_CREATE_PR`: Automatically create a Pull Request on completion.
- `AUTO_MERGE`: Automatically merge the Pull Request once checks pass.

### ActivityArtifact
- **Media**: `mimeType` (string), `data` (base64 string)
- **Bash Output**: `command` (string), `output` (string), `exitCode` (int)
- **Change Set**: `source` (string), `gitPatch` (object with `unidiffPatch` and `baseCommitId`)
