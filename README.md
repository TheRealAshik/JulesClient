# Jules Client Project

This repository contains the source code for the Jules Client application, separated into web and mobile platforms.

## Directory Structure

*   **`web/`**: The web application (React, Vite, TypeScript).
*   **`mobile/`**: The mobile application (Kotlin Multiplatform: Android, iOS, Desktop/JVM).

## Getting Started

Please refer to the `README.md` files within each directory for specific instructions on how to build and run each project.

*   [Web Application README](web/README.md)
*   [Mobile Application README](mobile/README.md)
